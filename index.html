Dragging with option
按著 option 並拖拉
Actually, if folder wants to be merged with another. The right way to do it is dragging souce folder to the destination and hold option key before you release dragging. It can be really frustrating and making mistake easily if you repeat again and again.
Especially, do it with touchpad. No Kidding!
事實上，若你想要在複製貼上時，將兩個相同名稱資料夾底下的檔案做合併，你可以拖拉來源資料夾，到目的資料夾上，然後在你放開拖拉前，先按著 option 即可。 要大量重複相同動作，卻沒辦法使用熱鍵來完成，這令人十分沮喪，而且拖拉的過程也容易出錯。
